# batman-adv
```
Batman advanced는 더 이상 IP 기반으로 작동하지 않는 무선 네트워킹에 대한 새로운 접근 방식이다. 

UDP 패킷을 사용하여 정보를 교환하고 라우팅 테이블을 설정하는 batman 데몬과 달리 batman-advanced는 ISO / OSI Layer 2에서만 작동하며 이더넷 프레임을 사용하고 라우팅한다 (또는 더 나은 방법 : 브리지).

참여하는 모든 노드의 가상 네트워크 스위치를 에뮬레이트한다. 

따라서 모든 노드가 링크 로컬 인 것처럼 보이므로 모든 상위 운영 프로토콜은 네트워크 내의 변경 사항에 영향을받지 않는다.
```
